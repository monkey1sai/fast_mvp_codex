## Full Profile

Hi, I'm building practical tools around MCP, email-based data ingestion, and LLM decision workflows.

This project, `pulse-ingestor-mvp`, focuses on a simple but useful problem:
turning scheduled ChatGPT email outputs into structured pulse events that downstream agents and LLM systems can actually use.

Your sponsorship helps me continue work on:

- maintaining and improving this repository
- writing clearer setup and usage documentation
- fixing bugs and edge cases in IMAP, mailbox handling, and MCP integration
- expanding the MCP tool surface for real agent workflows
- turning early MVP ideas into usable open-source infrastructure

If this project helps your workflow, your support directly increases the amount of time I can spend improving it.

If you need something more specific than sponsorship, I also offer implementation and advisory services for:

- MCP server design and packaging
- email ingestion pipelines
- LLM decision context systems
- FastAPI-based agent backends
- deployment and integration support
